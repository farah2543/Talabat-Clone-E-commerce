# Talabat Clone E-Commerce

A modern **E-Commerce web application** inspired by Talabat, built using **Angular** and modern frontend development practices.  
The project focuses on creating a scalable shopping experience with product browsing, cart management, authentication, and responsive UI design.

---

## Features

- User Authentication & Authorization
- Product Listing & Details
- Shopping Cart Management
- Responsive UI Design
- Product Filtering & Search
- Lazy Loading Modules
- Reusable Angular Components
- API Integration
- Route Guards
- Error Handling
- State Management Concepts

## Getting Started

### Prerequisites

Make sure you have installed:

- Node.js
- Angular CLI

---

## Installation

Clone the repository:

```bash
git clone -b Session_7 https://github.com/farah2543/Talabat-Clone-E-commerce.git
```

Navigate to the project folder:

```bash
cd Talabat-Clone-E-commerce
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
ng serve
```

The application will run on:

```bash
http://localhost:4200/
```

---

## Build

To build the project for production:

```bash
ng build
```
